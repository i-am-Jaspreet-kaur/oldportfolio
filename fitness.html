Design Study: Planet Fitness

Planet Fitness is an application mainly for the users who are enrolled as a member of the Planet Fitness gym while non-enrolled members can also use this app if they want to know about the club and various exercises.

Project Overview

The project was to do a design study on any mobile application that we use but do not like that much from the UX prospective, which was given by my professorfor the “Advanced User Experienec” class.

Problem

The app has many useful features but there is a lot of confusion while using the app as the user sometimes has to go deep enough to book a slot at the club and the icon for checking in the club is really confusing, it seems to be saved files. Also, there is redundancy in some features such as scan & learn inbox. There is a 'set up profile' button in the hamburger menu which is totally different from the name, it is all about the user's target or goals that they want to achieve from the gym.

User Flow and Wireframes

I created the new user flow solving the issues that I figured out to keep it simple and easy to use for the users. Below is the current user flow and the proposed user flow.

Wireframes

I created the wireframes using the user flow that I created. I tried to make it consistent with the current application as much as possible but simpler and easier to use. Below are the images of the original application along with the proposed wireframes.

Home Page of the application

  • The icon for check-in is changed and the name is added to let the users know what it is all about. Users can check in and book a slot in the club for any day and time.
  • "Refer" is replaced with "settings" which also includes "profile" which was earlier under the hamburger at the top left corner.
  • "Refer" is added to the home page at the bottom as a button.

Workouts Page

  • "Free workouts" is renamed as "Trainer's advice" to reduce the confusion for the user as it contains the trainer's videos to give advice to the users.
  • Back button for the user to make it easy to go back instead of pressing the 'workouts' icon again and then going back.
  • The search box is added to make it easy for the user to search for any video.
  • The filter button is placed at the top instead of the bottom. At the bottom, it disappeared when the user scrolls, now the user can easily press it than earlier.
  • The scroll bar is added to let users know that there are more videos at the bottom. .

Check-in Page

    Users can book a slot at the club beforehand on a date and time basis.
  • If the user wants to check in the club at the moment.
  • Users can click on the button to accept the guidelines and then it will open the QR code for the user which they can scan and then enter the club.
  • When the user will click on the icon, then the dates will pop up from the bottom, user can choose the date and time to book a slot at the club.
  • A message when there is no session or slot available on the day chosen by the user.
  • If the slot is not available at a particular day and club location then the user can find another club by clicking on it.
  • Users can select the date and time, a search bar to search for the club, and see the list of the locations of the club.
